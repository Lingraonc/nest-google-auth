## Description

[Nest.js](https://github.com/nestjs/nest) Simple auth with google and JWT token. Using refresh and access tokens.

## Installation

```bash
$ npm install
$ cp .env.example .env
```

## Run

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
